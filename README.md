# Тестовое задание для студии Перцы

Классический арканоид, реализованный на чистом JavaScript без использования сторонних игровых движков.  

## Описание 
Вы управляете платформой в нижней части экрана, разбиваете блоки в верхней части экрана, не давая мячу упасть.
Победа достигается при уничтожении всех блоков.
## Управление
Управление на клавиатуре, клавиша "A" - влево, "D" - вправо.
Чтобы начать игру нажмите пробел.

Ресурсы
- Textures: https://www.spriters-resource.com/arcade/arkanoid/asset/52829/
- Audio: https://sounds.spriters-resource.com/nes/arkanoid/asset/397226/

- На реализацию (с учетом правки ассетов): 5 часов
